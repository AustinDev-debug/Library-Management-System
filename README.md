# Library Management System

A C# console application that allows users to manage a small library.

## Features

- Add books
- List all books
- Search for books by title
- Check out books
- Return books

## Concepts Used

- Classes and Objects
- Lists (`List<T>`)
- Methods
- Loops
- Input Validation
- Switch Statements

## What I Learned

This project helped me learn how to:
- Store objects inside a `List<T>`
- Work with custom classes
- Separate functionality into methods
- Update object properties, such as whether a book is checked out

## Possible Future Improvements

- Delete books
- Edit book information
- Save the library to a file
- Load the library when the program starts
